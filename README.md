# 📘 คู่มือการใช้งานระบบ TPAC - Digital Score
## ระบบคะแนนพฤติกรรมนักเรียน โรงเรียนเทพา

---

## 🏠 สำหรับนักเรียน — `home.html`

หน้าแดชบอร์ดของนักเรียน แสดงคะแนนพฤติกรรมและเมนูต่างๆ

### เมนูหลัก

| ปุ่ม | ฟีเจอร์ | รายละเอียด |
|------|---------|-----------|
| 📊 **ประวัติ** | ดูการเคลื่อนไหวคะแนน | แสดงรายการหัก/เพิ่มคะแนนย้อนหลัง 3 เดือน พร้อมวันที่และเหตุผล |
| 📖 **กฎระเบียบ** | ดูเกณฑ์การตัดคะแนน | แสดงกฎทั้งหมด เช่น มาสาย -10, บุหรี่ไฟฟ้า -60 ฯลฯ |
| 📷 **QR Code** | แสดง QR Code ของตัวเอง | ใช้ให้ครูสแกนเพื่อค้นหาข้อมูลนักเรียน |
| 👤 **โปรไฟล์** | ดูข้อมูลส่วนตัว | ชื่อ, รหัสนักเรียน, สถานะพฤติกรรม |
| 🔔 **การแจ้งเตือน** | ติดตามสถานะคำร้อง | ดูว่าคำขอเพิ่มคะแนน/แจ้งเหตุของเราถูกอนุมัติหรือยัง |
| 📢 **แจ้งเหตุด่วน** | รายงานพฤติกรรม | แจ้งเหตุพฤติกรรมไม่เหมาะสมที่พบเห็น |

### วิธีขอเพิ่มคะแนน
1. กดปุ่ม **"กฎระเบียบ"**
2. เลื่อนลงกด **"ส่งขอเพิ่มคะแนน"**
3. เลือกประเภท **"ในโรงเรียน"** หรือ **"นอกโรงเรียน"**
4. กรอกรายละเอียดงาน/การแก้ไข
5. แนบรูป **ก่อนทำ** และ **หลังทำ**
6. กด **"บันทึกผลงาน"**

### คะแนนอัปเดตอัตโนมัติ
- ระบบจะดึงคะแนนจากเซิร์ฟเวอร์ทุก **3 วินาที**
- ถ้าคะแนนเปลี่ยน จะมี **Toast แจ้งเตือน** ปรากฏขึ้นทันที

## 👨‍🏫 สำหรับครูที่ปรึกษา — `th.html`

หน้าจัดการคะแนนนักเรียน สำหรับครูประจำวิชา/ครูที่ปรึกษา

### ปุ่มด้านบน (Action Buttons)

| ปุ่ม | ฟีเจอร์ | รายละเอียด |
|------|---------|-----------|
| ⌨️ **กรอกรหัส** | กรอกรหัสนักเรียนด้วยตนเอง | พิมพ์เลขรหัสแล้วกดค้นหา |
| 📸 **ถ่ายรูป QR** | ถ่ายภาพ QR จากกล้อง | เลือกรูปภาพ QR Code ที่ถ่ายไว้ |
| 📱 **สแกน QR** | เปิดกล้องสแกน QR แบบเรียลไทม์ | ส่องกล้องไปที่ QR Code โดยตรง |
| 🔍 **ค้นหา นร.** | ค้นหาจากรายชื่อทั้งหมด | ค้นหาตามชื่อ หรือ กรองตามชั้นเรียน |
| 📋 **รายงานสรุป** | ดูรายงานประจำเดือน | สรุปการหักคะแนน/ทำความดี กรองตามเดือน ห้อง ส่งออก PDF/CSV |
| 👔 **ตรวจแต่งกาย** | ตรวจสอบการแต่งกาย | เลือกห้อง/เดือน/รอบ แล้วเช็คเล็บ/ผม/เสื้อผ้า ส่งผลทีเดียว |

### วิธีหักคะแนนนักเรียน
1. ค้นหานักเรียนด้วยวิธีใดวิธีหนึ่ง (กรอกรหัส / สแกน QR / ค้นหา)
2. นักเรียนจะปรากฏในแถบ **"นักเรียนที่เลือก"**
3. กดปุ่ม **+** หรือ **-** เพื่อปรับคะแนน
4. เลือกหมวดหมู่เหตุผล (มาสาย, ทะเลาะ, ฯลฯ)
5. กด **"บันทึกข้อมูล"**

### วิธีดูรายงานสรุป
1. กดปุ่ม **"รายงานสรุป"** (ปุ่มสีเหลือง)
2. เลือก **เดือน** และ **ปีการศึกษา**
3. เลือกประเภท: **หักคะแนน** หรือ **ทำความดี**
4. กด **"ค้นหา"** เพื่อแสดงผล
5. สามารถกด **CSV** หรือ **PDF** เพื่อส่งออกรายงาน

### วิธีตรวจแต่งกาย
1. กดปุ่ม **"ตรวจแต่งกาย"** (ปุ่มสีฟ้า)
2. เลือก **ชั้นเรียน**, **เดือน**, **ปี**, และ **รอบที่ตรวจ** (ครั้งที่ 1 หรือ 2)
3. กด **"โหลดรายชื่อ"**
4. ติ๊กเครื่องหมายถูกที่ช่อง **เล็บ / ผม / เสื้อผ้า** สำหรับนักเรียนที่ผิดระเบียบ
5. กด **"ส่งผลตรวจ"** เพื่อบันทึกทั้งหมดในคราวเดียว

## 🛡️ สำหรับผู้ดูแลระบบ — `teacher-dashboard.html`

แดชบอร์ดสำหรับ Admin จัดการข้อมูลทั้งหมดของโรงเรียน

### ฟีเจอร์หลัก

| ฟีเจอร์ | รายละเอียด |
|---------|-----------|
| 📊 **สถิติภาพรวม** | แสดงจำนวนนักเรียนทั้งหมด, จำนวนที่ถูกหักคะแนน, สถานะเสี่ยง/ปรับปรุงด่วน |
| 📋 **รายการรอตรวจสอบ** | ดูรายการคำขอจากนักเรียน/ครู ที่รอการอนุมัติหรือปฏิเสธ |
| ✅ **อนุมัติ/ปฏิเสธ** | กดอนุมัติเพื่อเพิ่มคะแนน หรือปฏิเสธพร้อมระบุเหตุผล |
| 🔍 **ค้นหานักเรียน** | ค้นหาข้อมูลนักเรียนรายบุคคล ดูคะแนนและประวัติ |
| 📈 **รายงานสรุป** | ดูสถิติการหักคะแนน/ทำความดี แยกตามห้อง/เดือน ส่งออก PDF/CSV |
| 🏆 **รายชื่อทำความดี** | ดูรายชื่อนักเรียนที่ส่งผลงานทำความดี ทั้งในและนอกโรงเรียน |

### วิธีอนุมัติคำขอ
1. ดูรายการใน **"รายการรอตรวจสอบ"**
2. คลิกที่รายการเพื่อดูรายละเอียด (รูปภาพ, รายละเอียดงาน)
3. กด **"อนุมัติ"** เพื่อเพิ่มคะแนนให้นักเรียน
4. หรือกด **"ปฏิเสธ"** พร้อมระบุเหตุผล

### วิธีดูรายงาน
1. เลือก **ปีการศึกษา** และ **เดือน** จากตัวกรอง
2. เลือก **ห้องเรียน** ถ้าต้องการกรองเฉพาะห้อง
3. คลิกที่ปุ่ม **สรุปรายงาน** เพื่อดูรายละเอียดทั้งหมด
4. กด **PDF** หรือ **CSV** เพื่อดาวน์โหลดรายงาน

## 🔐 ระบบเข้าสู่ระบบ — `index.html`

| Role | ข้อมูลที่ต้องกรอก | หน้าที่เข้า |
|------|------------------|------------|
| 🎓 **นักเรียน** | รหัสนักเรียน + รหัสผ่าน | `home.html` |
| 👨‍🏫 **อาจารย์** | ชื่อผู้ใช้ + รหัสผ่าน | `th.html` |
| 🛡️ **ผู้ดูแล** | ชื่อผู้ดูแล + รหัสผ่าน | `teacher-dashboard.html` |
| 📋 **กรรมการ** | รหัสคณะกรรมการ + รหัสผ่าน | `cd.html` |

### ระบบจดจำการเข้าสู่ระบบ (Cookie)
- เมื่อ Login สำเร็จ ระบบจะจำไว้  
- **ครั้งต่อไปเปิดเว็บ** → เข้าสู่ระบบให้อัตโนมัติ โดยไม่ต้อง Login ใหม่  
- **กดออกจากระบบ** → ระบบจะลืม ครั้งต่อไปต้อง Login ใหม่

## 📦 ไฟล์สำคัญในระบบ

| ไฟล์ | หน้าที่ |
